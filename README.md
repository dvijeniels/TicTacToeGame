# TicTacToeGame
Игра в крестики нолики. Где можно выбирать с кем будешь(С другом или с компом) играть и поставить таймер. Tic Tac Toe oyunu. Ouyun başlamadn önce kimle oynayacağını seçiyorsun ve zaman verebilirsiniz.
